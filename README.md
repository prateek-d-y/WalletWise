# WalletWise Expense Tracker

WalletWise is a beautiful, fully responsive, and simple expense tracker built directly inside the browser using HTML, CSS, JavaScript, and Local Storage.

## Features
- **Offline First**: All your financial data is saved locally on your device via browser Local Storage. No database setup required.
- **Responsive UI**: Looks great on both Desktop and Mobile phones. Bottom tab navigation for mobile users.
- **Dark Mode**: Instantly switch between vibrant light mode and sleek dark mode.
- **Analysis Dashboard**: Built-in Chart.js pie charts and line graphs to visualize your spending.
- **Multiple Accounts**: Manage separate balances for Cash, Bank Accounts, Wallets, etc. Automatic deduction upon recording expenses.

## How to Run Locally
1. Clone or download this folder.
2. Double-click `index.html` to open it in your web browser.
3. Start adding accounts and tracking expenses!

## File Structure
- `index.html`: The main Single Page Application structure.
- `css/style.css`: All styling, CSS variables for theming, and responsive media queries.
- `js/app.js`: State management, DOM manipulation, Modal control, and Chart initialization.
